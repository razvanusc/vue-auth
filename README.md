# Vue Authentication

## Getting started

### Installing

These instructions will get you a copy of the project up and running on your local machine.

Install missing packages
```
npm install
```

Run local development server
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
